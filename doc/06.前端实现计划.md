## 一、设计令牌体系（Design Tokens）— 主题色可全局切换方案

### 1.1 核心思路

将所有颜色、间距、圆角、字号等设计变量提取为 CSS 自定义属性（CSS Variables），集中定义在 `:root` 中。修改主题色只需改一处，全站自动生效。

### 1.2 颜色令牌定义

```css
:root {
  /* ===== 主题色（修改这3个值即可切换全站主题） ===== */
  --color-primary: #008c8c;
  --color-primary-hover: #00a8a8;
  --color-primary-active: #007575;
  --color-primary-bg: #e6f7f7;       /* 主色浅底（active菜单、选中态） */
  --color-primary-light: rgba(0,140,140,0.15); /* 主色半透明 */

  /* ===== 功能色（语义色，一般不随主题变） ===== */
  --color-success: #00b87a;
  --color-success-bg: #e8f8f2;
  --color-info: #1890ff;
  --color-info-bg: #e8f4ff;
  --color-warning: #fa8c16;
  --color-warning-bg: #fff7e6;
  --color-danger: #ff4d4f;
  --color-danger-bg: #fff1f0;
  --color-purple: #722ed1;
  --color-purple-bg: #f9f0ff;

  /* ===== 中性色（灰阶体系） ===== */
  --color-text-primary: #26272a;      /* 主文本 */
  --color-text-secondary: #4e5464;    /* 次文本 */
  --color-text-tertiary: #8a8b8e;     /* 辅助文本 */
  --color-text-placeholder: #999;     /* 占位符 */
  --color-text-disabled: #d9d9d9;     /* 禁用态 */

  --color-bg-page: #f5f7fa;           /* 页面背景 */
  --color-bg-card: #ffffff;           /* 卡片/容器背景 */
  --color-bg-hover: #f5f7fa;          /* hover背景 */
  --color-bg-table-header: #fafafa;   /* 表头背景 */
  --color-bg-disabled: #f5f5f5;       /* 禁用背景 */

  --color-border: #ecedf0;            /* 主边框 */
  --color-border-input: #d9d9d9;      /* 输入框边框 */
  --color-border-split: #ecedf0;      /* 分割线 */

  /* ===== 特殊业务色 ===== */
  --color-accent: #e6a23c;            /* 强调按钮（舰队管理等） */

  /* ===== 间距体系 ===== */
  --spacing-xs: 4px;
  --spacing-sm: 8px;
  --spacing-md: 12px;
  --spacing-lg: 16px;
  --spacing-xl: 20px;
  --spacing-xxl: 24px;

  /* ===== 圆角 ===== */
  --radius-sm: 2px;    /* 标签 */
  --radius-md: 4px;    /* 标准组件 */
  --radius-lg: 8px;    /* 大卡片/登录框 */
  --radius-full: 50%;  /* 圆形 */

  /* ===== 字号 ===== */
  --font-size-xs: 11px;
  --font-size-sm: 12px;
  --font-size-base: 14px;
  --font-size-md: 15px;
  --font-size-lg: 16px;
  --font-size-xl: 18px;
  --font-size-xxl: 20px;

  /* ===== 阴影 ===== */
  --shadow-sm: 0 2px 8px rgba(0,0,0,0.08);
  --shadow-md: 0 4px 12px rgba(0,0,0,0.12);
  --shadow-lg: 0 8px 32px rgba(0,0,0,0.15);

  /* ===== 布局尺寸 ===== */
  --layout-max-width: 1440px;
  --layout-min-width: 1080px;
  --sidebar-width: 15%;
  --sidebar-max-width: 265px;
  --sidebar-min-width: 180px;
  --header-height: 48px;
  --breadcrumb-height: 40px;
}
```

### 1.3 使用方式

所有组件样式中引用变量而非硬编码颜色值：

```css
/* 示例：按钮 */
.btnPrimary {
  background: var(--color-primary);
  color: #fff;
  border: none;
}
.btnPrimary:hover { background: var(--color-primary-hover); }
.btnPrimary:active { background: var(--color-primary-active); }

/* 示例：侧边栏 */
.sidebarLogo { background: var(--color-primary); }
.menuItem.active {
  background: var(--color-primary-bg);
  color: var(--color-primary);
  border-left-color: var(--color-primary);
}

/* 示例：Header */
.header { background: var(--color-primary); }
```

### 1.4 主题切换方案

方案一：直接修改 `:root` 变量值（最简单，适合固定主题）

方案二：通过 `data-theme` 属性切换预设主题

```css
/* 默认主题（青色） */
:root { --color-primary: #008c8c; ... }

/* 蓝色主题 */
[data-theme="blue"] {
  --color-primary: #1890ff;
  --color-primary-hover: #40a9ff;
  --color-primary-active: #096dd9;
  --color-primary-bg: #e6f7ff;
}

/* 紫色主题 */
[data-theme="purple"] {
  --color-primary: #722ed1;
  --color-primary-hover: #9254de;
  --color-primary-active: #531dab;
  --color-primary-bg: #f9f0ff;
}
```

切换方式：
```javascript
document.documentElement.setAttribute('data-theme', 'blue');
```

## 二、分辨率适配方案

### 2.1 设计基准

- 设计稿基准：1440×900（主流笔记本）
- 最大宽度：1440px（超出居中显示）
- 最小宽度：1080px（低于此出现横向滚动条）
- 不使用响应式断点（media query），使用百分比 + min/max 约束

### 2.2 各分辨率表现

| 分辨率范围 | 表现 |
|---|---|
| ≥1440px | 内容区居中，两侧留白 |
| 1080px ~ 1440px | 内容区 100% 宽度，侧边栏和内容按比例缩放 |
| <1080px | 出现横向滚动条（min-width: 1080px 兜底） |

### 2.3 布局骨架适配规则

```css
.pageContainer {
  width: 100%;
  max-width: var(--layout-max-width);  /* 1440px */
  min-width: var(--layout-min-width);  /* 1080px */
  margin: 0 auto;
  display: flex;
  height: 100vh;
}

.sidebar {
  width: var(--sidebar-width);          /* 15% */
  max-width: var(--sidebar-max-width);  /* 265px */
  min-width: var(--sidebar-min-width);  /* 180px */
  flex-shrink: 0;
}

.mainArea {
  flex: 1;
  min-width: 0;  /* 防止flex子项溢出 */
}
```

### 2.4 表格适配

- 表格容器使用 `overflow-x: auto`，窄屏时表格可横向滚动
- 列宽使用固定 px + 弹性列（不设宽度的列自动填充）
- 操作列固定宽度（120px~200px），右对齐

### 2.5 查询区适配

- 使用 `flex-wrap: wrap`，窄屏时查询项自动换行
- 每个查询项固定宽度（label + 180px input）
- 查询按钮组使用 `margin-left: auto` 靠右

### 2.6 弹窗适配

- 弹窗使用固定宽度（420px~620px），居中显示
- 弹窗最大高度 80vh，内容区 overflow-y: auto
- 窄屏时弹窗宽度不超过 `calc(100vw - 32px)`

## 五、主题切换实现步骤

### 步骤 1：提取令牌

将 `prototype-layout.css` 中所有硬编码颜色值替换为 CSS 变量引用。

### 步骤 2：创建主题文件

每个主题文件只需覆盖 `--color-primary` 系列变量：

```css
/* themes/blue.css */
:root[data-theme="blue"] {
  --color-primary: #1890ff;
  --color-primary-hover: #40a9ff;
  --color-primary-active: #096dd9;
  --color-primary-bg: #e6f7ff;
  --color-primary-light: rgba(24,144,255,0.15);
}
```

### 步骤 3：运行时切换

```javascript
// 切换主题
function setTheme(theme) {
  document.documentElement.setAttribute('data-theme', theme);
  localStorage.setItem('app-theme', theme);
}

// 页面加载时恢复主题
const savedTheme = localStorage.getItem('app-theme');
if (savedTheme) setTheme(savedTheme);
```

## 七、实施注意事项

1. **渐进迁移**：先建立 tokens.css，再逐步将各组件中的硬编码颜色替换为变量引用
2. **向后兼容**：CSS 变量不支持 IE，如需兼容可使用 PostCSS 插件做降级
3. **组件隔离**：每个组件的样式文件只引用变量，不直接写颜色值
4. **暗色主题预留**：当前只做浅色主题，但变量体系天然支持后续扩展暗色主题
5. **币种显示**：所有币种字段使用中文全称（美元、人民币），不使用 ISO 缩写
6. **操作术语**：统一使用"置无效/置有效"，禁止使用"删除/恢复"
